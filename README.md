🔍 Screen Magnifier – Экранная лупа на 7 языках
Увеличивайте любой фрагмент экрана прямо в терминале!
Мощная утилита для захвата и увеличения области изображения с цветным выводом, поддержкой нескольких форматов и гибкими настройками.
Идеально подходит для демонстраций, презентаций и помощи людям с ослабленным зрением.
Реализована на 7 языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Загрузка изображений – поддерживаются форматы PNG, JPG, BMP, GIF.

Выбор области – укажите координаты (x, y) и размер (ширина, высота) в пикселях.

Увеличение – настраиваемый коэффициент масштабирования (от 2 до 20).

Цветной вывод – каждый пиксель отображается цветным блоком с использованием ANSI-кодов.

Адаптивная яркость – автоматическая регулировка яркости для улучшения читаемости.

Детализация – показ координат и размеров увеличенной области.

Поддержка больших изображений – эффективная работа с файлами любого размера.

Кросс-платформенность – работает в любом терминале с поддержкой ANSI-цветов.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> <файл_изображения> -x <X> -y <Y> -w <Ширина> -h <Высота> -s <Коэффициент> [опции]
Аргументы
Опция	Описание
<файл>	Путь к изображению (обязательно)
-x <X>	Координата X верхнего левого угла области (по умолчанию 0)
-y <Y>	Координата Y верхнего левого угла области (по умолчанию 0)
-w <W>	Ширина области в пикселях (по умолчанию 40)
-h <H>	Высота области в пикселях (по умолчанию 20)
-s <N>	Коэффициент увеличения (по умолчанию 4)
--auto	Автоматически выбрать область вокруг курсора (если поддерживается)
--no-color	Отключить цветной вывод
--help	Справка
Примеры
bash
# Увеличить область 40×20 пикселей в центре изображения в 4 раза
python magnifier.py screenshot.png -x 200 -y 150 -w 40 -h 20 -s 4

# Увеличить с коэффициентом 8 и отключить цвет
python magnifier.py photo.jpg -x 100 -y 100 -w 30 -h 30 -s 8 --no-color

# Автоматический режим (если реализован)
python magnifier.py screenshot.png --auto
🛠 Установка и запуск
Python
bash
pip install pillow
python magnifier.py <файл> [опции]
Требуется Python 3.6+ и библиотека Pillow.

C++
bash
sudo apt-get install libopencv-dev libjsoncpp-dev  # для Ubuntu
g++ -std=c++17 magnifier.cpp -o magnifier `pkg-config --cflags --libs opencv4`
./magnifier <файл> [опции]
Требуется OpenCV 4+.

Go
bash
go get github.com/disintegration/imaging
go build magnifier.go
./magnifier <файл> [опции]
Требуется Go 1.16+ и пакет imaging.

JavaScript (Node.js)
bash
npm install sharp
node magnifier.js <файл> [опции]
Требуется Node.js 12+ и пакет sharp.

C#
bash
dotnet add package SixLabors.ImageSharp
csc magnifier.cs
mono magnifier.exe <файл> [опции]   # или dotnet run
Требуется .NET Core 3.1+.

Ruby
bash
gem install rmagick
ruby magnifier.rb <файл> [опции]
Требуется Ruby 2.5+ и rmagick.

Java
bash
javac -cp .:gson.jar magnifier.java
java -cp .:gson.jar magnifier <файл> [опции]
Требуется Java 11+ и библиотека gson для JSON (опционально).

🧠 Логика работы
Загрузка изображения – чтение файла в память.

Извлечение области – копирование пикселей из заданного прямоугольника (x, y, width, height).

Масштабирование – увеличение области до размера (width * scale) × (height * scale) с использованием билинейной интерполяции.

Вывод – отображение каждого пикселя в виде цветного квадрата (или символа) в терминале.

Дополнительно – вывод информации о координатах, размерах и цветовой палитре.

✨ Дополнительные фичи
Интерполяция – плавное увеличение без пикселизации (билинейная или ближайшего соседа).

Адаптивная яркость – автоматическая коррекция контраста для улучшения видимости.

Палитра – использование 256-цветной палитры терминала для точного отображения цветов.

Экспорт – сохранение увеличенной области в файл (опция --output).
